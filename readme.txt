== Installation ==

Install New Google Plus Badge either via the WordPress.org plugin directory or by uploading the files to your server.

After activating New Google Plus Badge Widget plugin, go to plugin menu.

Login into admin dashboard. Go to Appearance --> Widgets : New Google+ Badge

Activate the New Google Plus Badge Widget into widget area of the theme.

Configure settings and save.

That's it. You're ready to go!
